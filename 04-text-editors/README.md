Vim editor usage and commands:
:w for save
:q quit 
:wq save n exit
:q! discard quit
x at normal mode deleting word that under cursor
a for append to cursors right
dw for delete word starts from cursor deletes blanks too till other words 
de only for delete that word not blanks and to other word just all characters of that word
d$ delete end of the line
w for move cursor to next word and multiplies with numbers like 2w 
e for move cursor to end of line and multiplies with number like 3e
0 zero to move cursor start of the line
there is a combination chance like
d2w  delete  and move other word like 2 times 
dd delete whole line can be combined like 2dd deletes 2 whole lines
u for undo U for fix whole line if u hit second time u redos
ctrl+r redos undo the undos
